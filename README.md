# Repositori Tes Aplikasi Pekerjaan

Repositori ini dibuat untuk memenuhi persyaratan tes aplikasi pekerjaan.

## Deskripsi

Proyek ini adalah aplikasi [Next.js](https://nextjs.org) yang dibuat dengan [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Memulai

Jalankan server pengembangan:

```bash
npm run dev
# atau
yarn dev
# atau
pnpm dev
# atau
bun dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser Anda untuk melihat hasilnya.

## Informasi Lebih Lanjut

Untuk mempelajari lebih lanjut tentang Next.js, kunjungi:

- [Dokumentasi Next.js](https://nextjs.org/docs) - pelajari fitur dan API Next.js.
- [Belajar Next.js](https://nextjs.org/learn) - tutorial interaktif Next.js.

## Deploy di Vercel

Cara termudah untuk mendeploy aplikasi Next.js Anda adalah menggunakan [Platform Vercel](https://vercel.com/new?utm_medium=default-template&filter=next.js&utm_source=create-next-app&utm_campaign=create-next-app-readme) dari kreator Next.js.
